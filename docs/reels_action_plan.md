---

## 🎬 動画制作チェックリスト（毎回）

- [ ] **7〜15秒**に収める（視聴完了率が最重要指標）
- [ ] **最初の1〜3秒**にフックを入れる
- [ ] **1080p以上**の画質で書き出す
- [ ] BGMを設定する（音量50〜70%）
- [ ] カバー画像を設定する

### 動画構成テンプレート

```
[0〜1秒]  テキストフック
[1〜5秒]  ビフォーアフター or ルームツアー
[5〜10秒] こだわりポイントのクローズアップ
[10〜15秒] 全体ショット + CTA（「保存してね！」）
```

## 🎵 音源戦略

**お気に入り2〜3曲 + トレンド音源の併用**

### お気に入りに選ぶジャンル
- Lo-fi / チル系（落ち着いたルームツアー向け）
- アコースティック（ナチュラルテイスト向け）
- ピアノBGM（高級感ある部屋向け）

### ローテーション例
| 投稿 | 音源 |
|:---:|---|
| 1 | 🎵 お気に入り① |
| 2 | 🎵 お気に入り② |
| 3 | 🔥 トレンド音源 |
| 4〜 | 繰り返し |

### 注意点
- **歌詞なし（インスト）推奨** — 映像に集中させる
- **テンポ速すぎない曲** — 保存率UP
- **音量は50〜70%** — おしゃれ感UP
- トレンド音源は**流行り始めて1〜2週間以内**に使う
