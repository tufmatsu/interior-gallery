# Instagram キャプションテンプレート

## 構成

```
① フック（1行目）
② 内容（2〜4行）
③ CTA（1行）
④ ハッシュタグ（5〜15個）
```

---

## テンプレート（コピペ用）

```
（フック：興味を引く1行）

（何畳の部屋か）
（どんな人向けか / こだわりポイント1〜2個）

→ 使用アイテムはプロフのリンクからチェック🔗
→ 保存しておくとあとで見返せるよ📌
→ 「こんな部屋見たい！」があればコメントで教えてね🙋‍♂️
※CTAは毎回2つ選んでローテーション

#一人暮らし #一人暮らしインテリア #○畳レイアウト
#ルームツアー #インテリアコーディネート
#（スタイル系タグ） #（アイテム系タグ）
#お部屋づくり #3Dインテリア
#ひとへやLab
```

---

## ① フックのパターン集

- 「○○してみた」系
  → 「木目の家具で"落ち着く部屋"を作ってみた🪵」
- 「○○が正解だと思う」系
  → 「狭い部屋こそ、丸テーブルが正解だと思う」
- 「○○のために○○した」系
  → 「6畳でもデスク環境を諦めない方法🖥️」
- 数字系
  → 「6.2畳に"最強のデスク環境"を作った結果」

---

## ③ CTAの使い分け

| 目的 | 文 |
|---|---|
| サイトへ誘導 | 「使用アイテムはプロフのリンクから🔗」 |
| 保存を促す | 「参考になったら保存してね📌」 |
| コメントを促す | 「似た部屋に住んでる人いる？🙋」 |
| リクエスト募集 | 「こんな部屋見たい！があればコメントで教えてね🙋‍♂️」 |
| リクエスト（選択肢付き） | 「次はどんな部屋が見たい？👇 🛏️8畳ワンルーム / 🖥️ゲーマー部屋 / 🌿北欧ナチュラル」 |

※毎回2つ選んでローテーションするのがおすすめ（全部入れると長い）
